## Getting Started

### Pré-requis

* npm
```sh
npm install npm@latest -g
```

### Installation

1. Clonez le dépôt
```sh
git clone https://github.com/khylias/enquete-dev.git
```
2. Installez les dépendances
```sh
npm install
```
3. Lancez le serveur de dev 
```sh
npm run serve
```
